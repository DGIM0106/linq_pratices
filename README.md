En este repositorio se puede revisar el avance por linq

Método                      Propósito     ¿Qué hace en la tubería?
.Where()                    Filtrar       Deja pasar solo los elementos que cumplen una condición.
.Select()                   Transformar   Cambia el tipo de dato o extrae una propiedad específica.
.OrderBy()/Descending       Ordenar       Organiza los elementos bajo el criterio que elijas.
.GroupBy()                  Agrupar       Organiza la lista en ""cajones"" basados en una clave (.Key)."
.Sum()/.Average()           Calcular      Realiza operaciones matemáticas sobre las propiedades de la lista.
.First()/.FirstOrDefault()  Extraer       Obtiene un único elemento de forma segura.
